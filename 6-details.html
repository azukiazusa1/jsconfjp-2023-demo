Help

What is this?

This is a simple web application that allows you to create and manage your own personal todo list.

How do I use it?

You can create a new todo by clicking the "New Todo" button. You can delete a todo by clicking the "Delete" button. You can mark a todo as complete by clicking the checkbox next to it. You can edit a todo by clicking the "Edit" button.

How do I create a new todo?

Click the "New Todo" button. Enter a title and description for your todo. Click the "Save" button.
